Frequently Asked Questions

Quick answers to common questions about Core Konstruct.

Core Konstruct is a role-based construction monitoring platform designed for contractors, supervisors, and clients. It provides real-time project tracking, daily reporting, expense management, attendance monitoring, and photo documentation — all in one dashboard.
The platform supports three user roles: Admin (Contractor) for overall project management, Supervisor for site-level reporting and attendance, and Client for viewing project progress and updates. Each role has a tailored dashboard.
Yes. Core Konstruct uses JWT-based authentication with encrypted passwords (bcrypt hashing). Each user only sees data relevant to their role, ensuring privacy and security across all dashboards.

When supervisors submit daily reports with expenses (labour, material, miscellaneous), the system automatically aggregates these into the project's finance panel. Admins can see budget vs actual spending with category-wise breakdowns in real time.
