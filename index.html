❓ FAQ

Frequently Asked Questions

Everything you need to know about Core Konstruct.

Project progress is tracked stage-by-stage using a Work Breakdown Structure (WBS). Each supervisor updates the percentage completion for their assigned stage daily. The system automatically calculates the overall project progress as a weighted average of all stages. Contractors see a live dashboard with circular progress rings and stage trackers.
Clients receive a secure login to their dedicated Client Dashboard. They can view real-time progress bars, current construction stage, a photo gallery of daily site photos, and a project timeline with milestones. The client view is deliberately simplified — no technical or financial data is shown, keeping it clean and easy to understand.
Core Konstruct supports Buildings (residential & commercial), Roads & Highways, Bridges, Drainage & Pipeline projects, and custom project types. Each project type has a tailored measurement module — for example, road projects track layer thickness and length completed, while building projects track slab area and column heights.
Absolutely. The Supervisor Dashboard is built with a mobile-first approach — large buttons, simple forms, and an easy photo upload interface designed for use on a construction site. Supervisors can mark attendance, submit daily reports, enter material usage, and upload photos all from their smartphone.
Yes. The Contractor/Admin Dashboard supports unlimited projects. You can create new projects, select the project type, define work stages, assign a supervisor, and set the budget. All active projects are displayed with progress rings, delay indicators, and financial summaries so you control everything from one screen.
The finance module tracks Labour Cost, Material Cost, and Miscellaneous Expenses separately for each project. The admin dashboard shows a cost breakdown with animated bars, compares actual spending against the project budget, and highlights if costs exceed safe thresholds. Financial data is only visible to admin — supervisors and clients cannot see financial details.
